CG-Invoice Instructions and Tips

Create an Invoice

To create a new invoice open menu item "Open Blank Invoice." fill out data and use "Submit" green button to save. Do not forget to set the date filed so you can keep acurate track of appended invoices.

Edit an Invoice

For security, no invoice can be deleted. You may however Clone an invoice by using the Edit button.

Clone will make a copy of the existing invoice and allow you to edit the following fields:

  • Invoice Number
  • Description
  • Cost
  • Sub
  • Total Total

Other fields are locked so you may append current customer invoice without losing vital information.

Administrator Information

Adding a new user requires a username or an email (both is best). Add the new username or email to the fields in admin page "Manage Access." At this point the new user may enter those credentials you just gave them into the fields of the pre-registration form---on the normal login page. Once this is done the new user will find they now have access to the registration form and can create and control their new account on their own. The new user does not have to use the email or the username you just gave them. You are basically creating a token or secret access key which will have to match when they enter it into the access registration form fields.

"Manage Accounts" page give administrators the ability to delete accounts and change username or password. There is also a field for active and non-active if you just want to put an account on hold without deleting it. A 1 in the active box will give user access and a 0 will lock them out.

To create a new administrator you can assign any new user---or current user---a "Level" of zero. Access levels are as such: 1 = regular user (no admin access); 0 = adminstrator access. Without a level of 0 users will not be able to access any administrative pages. "Manage Accounts" page gives you the options to change this freely.

Email field in admin can not be changed. Email is required to validate an account or make any changes. To change email you would have to create a new account with the new email.
